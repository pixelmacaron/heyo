Learn HTML and CSS

HTML

Headings

HTML headings can help you make titles or subtitles. There are six types of headings, <h1> being the biggest, and <h6> being the smallest. Because of the variety, they can be useful. To test them out yoursef, click here.

Paragraphs

Parapraphs are used to write text, or the main content. You can write them between the <p> and </p> tags. If you want to learn more, click here.

Text Formatting

With text formatting, there are a lot of options. You can add bold text with <b> tags, italic text with <i> tags, underlined text with <ins> tags, deleted text with <del> tags, superscript text with <sup> tags, subscript text with <sub> tags, and marked text with <mark> tags. Try it out over here.

Links

With HTML links, you can link different websites. Most of the time, links are text, but you can also set an image as a link. To use one, you just need to write <a href= "Your link here">Your link's name</a> You can link to any website, with this tag. If you want to test it out yourself, click here (it is a link!).

Images

With HTML images, you can show images on a webpage! It can be any type of image. On the home part of my webpage, I included a gif, which works too. You can either link it directly from a website, or show it because you downloaded it. Test them out here.

Lists

With HTML lists, you can make both ordered and unordered lists. Ordered lists go in a 1, 2, 3, pattern, while the items on unordered lists are separated by a bullet point. For an ordered list, you should use <ol>, and for an unordered list, use <ul>.Test them both over here.

Emojis

With HTML emojis, you can dispay emojis! There are all types of emojis. They each have a special code, and if you type it, you will get the emoji. In order to use them, type &#, their number, and ;. Find a list of all the emojis you can use here. ☕

Audio

With the new HTML5 audio feature, you can record or donwnload audio, and put it on a webpage. You can put audio that you recorded yourself, or found somewhere and downloaded. It is new to HTML5, so it might not work on some old browsers. However, if you want to, you can try it here.

Videos

With the new HTML5 video feature, you can record or donwnload videos, and put it on a webpage. Just like the audio, you can put videos that you recorded yourself, or found somewhere and downloaded. It is also new to HTML5, so it might not work on some old browsers. You can try it here.

CSS

Colors

With CSS colors, you can pick from a variety from colors. They can be used for text, borders, background colors,and even more! You have to specify what it is, and then say the name or HEX value. Find a list of the offical colors here.

You can aply the CSS colors to HTML objects! You just need to specify what it is. You can specify the HEX or color name. Try it out here, here, or here.

Fonts

With CSS, you can change fonts! Many webpages use this feature. There are many fonts that you can use, and there are even font combinations. Try out the fonts over here, or see a list of fonts here.

Borders

You are able to add borders with CSS! There are a variety of options, and they all can be helpful when styling borders. By deafault, they are black, but you can also change their color with the CSS color property. Click here to try them all out.

Text Effects

With CSS text effects, you can add effects to your text and pictures, such as shadows and outlines. This helps make your text unique. See and try them all here.

Align

With CSS, you can even align text or images. This can be done with the margin element. You are able to set the margin. This can be done with classes (which I will explain later), or directly in the HTML code. If you want to see an example, click here.

Classes

When you use classes, you can set what an element or part of text looks like. It can be usefull, if you want to set a some styles to an element, but don't want it with all elements. You can set multiple classes at once, at once, by seperating with using spaces. Classes can be very helpful, and are used ofthen with coding with HTML and CSS. here.

Padding

CSS allows to to add padding to items. This can help items like images, borders, and boxes of text stay far away from each other. When you are writing text inside of a box, then this can control how much space the text has, and how far apart everything is spaced. See how it works here.
